# HMI GCDC 2016

## Build and run
### Server
Start in root directory.
````
cd server
npm install
npm start
````
### Client
Open a new terminal tab and cd back into root.
````
cd client
npm install
npm run dev
````

HMI is now running on [localhost:8080](http://localhost:8080/).
